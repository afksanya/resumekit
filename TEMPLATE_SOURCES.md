# Template Sources

ResumeKit's built-in templates are original, dependency-free HTML/CSS layouts. They are not copied from third-party template files.

The expanded template set was informed by common patterns in open-source resume projects:

| Source | License | How ResumeKit uses it |
| --- | --- | --- |
| [Reactive Resume](https://rxresu.me/) | MIT | General inspiration for offering multiple visual resume styles. |
| [JSON Resume themes](https://docs.jsonresume.org/themes) | Mostly MIT/ISC theme ecosystem | Theme naming and print-friendly structured resume ideas. |
| [html-resume](https://github.com/mnjul/html-resume) | Apache-2.0 | Inspiration for browser-printable single-page HTML resumes. |
| [Start Bootstrap Resume](https://github.com/StartBootstrap/startbootstrap-resume) | MIT | Inspiration for sidebar-style resume layouts. |

If ResumeKit later copies or adapts specific third-party code, keep the original license notice and attribution in this file.
